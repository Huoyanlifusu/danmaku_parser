[info] <br>
基于Web Socket通信，发包至Bilibili直播间，抓取任意直播间直播弹幕信息

声明：本库仅限于学习使用，任何人非法使用造成的不良后果与本人无关。

[Usage] <br>
`pip install .`

[Application] <br>

+ demo
    + demo1 --- 客户端弹幕控制服务主机键盘（比如发送'111' - 按下E键）
    + demo2 --- 一个简易的弹幕姬实现（包括背景色、透明度调整） 当前有bug，很多弹幕信息无法计入，怀疑是解析问题，待解决

弹幕互动、流量统计（待开发demo）
